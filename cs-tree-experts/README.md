# CS & S Tree Experts Website

Professional tree service website built with Next.js 14, React, and Tailwind CSS.

## Features

- 🎨 Modern, responsive design
- 📱 Mobile-first approach
- ⚡ Fast performance with Next.js App Router
- 🌳 Custom tree service branding
- 📝 Contact form
- ⭐ Reviews section
- 📍 Service area display
- 🔧 Multiple service listings

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Building for Production

```bash
npm run build
npm start
```

## Customization

- Update business info in `src/app/layout.tsx` (metadata)
- Update contact details in `src/app/page.tsx` (phone, address)
- Add your own images to `/public` folder
- Modify services array in `src/app/page.tsx`
- Update Google Maps link in reviews section
